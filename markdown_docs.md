# Multistep Transcriber Documentation

This documentation provides an overview of the multistep-transcriber project's modules and their functionality.

## Overview

The multistep-transcriber project is designed to transcribe audio/video content through a series of processing steps. It consists of a main `VideoTranscriber` class that orchestrates various processing modules.

## Main Components

### VideoTranscriber Class
The `VideoTranscriber` class is the central component that manages the transcription pipeline. It handles:
- Audio extraction from video files
- Step-by-step processing of audio segments
- Integration with external services (Whisper, Gliner, etc.)

### Processing Steps

The project includes several processing modules that handle specific aspects of the transcription workflow:

1. **transcription.py** - Handles audio transcription using Whisper
2. **merge_sentences.py** - Merges transcribed sentences into coherent segments
3. **entities.py** - Identifies named entities in the transcribed text
4. **standardize.py** - Standardizes text formatting
5. **diarization.py** - Performs speaker diarization
6. **topic_segmentation.py** - Segments content by topics
7. **format.py** - Formats output for various destinations
8. **helpers.py** - Provides utility functions for processing
9. **merge_sentences.py** - Merges transcribed sentences into coherent segments

## Installation and Setup

The project requires Python 3.8+ and several dependencies. See the README.md for installation instructions.

## Usage

The main entry point is the `VideoTranscriber` class, which can be instantiated and used to process video files through the various processing steps.
